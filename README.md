# Equity & Efficiency in Patient Care 

## Objective

This project visualises and analyses healthcare practice data to identify gaps in care delivery, treatment delays, and demographic disparities. The goal is to highlight opportunities for more equitable and efficient patient care across populations.

## Technologies

* **Data Source:** CSV practice data (Tech Native Digital) 
* **Language:** SQL
* **Visualisation:** Google Sheets

## Data Architecture

* Data is extracted from CSV files
* Data is transformed and joined using SQL
* Results are visualised in Google Sheets
